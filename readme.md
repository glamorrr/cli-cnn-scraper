# Scrape news from CNN Indonesia

```
node . "SEARCH_VALUE" TOTAL_SEARCH_PAGES

// example
node . "pemilu 2024" 2
> [scraping] crawling 2 pages to collect articles related to pemilu 2024...
> [scraping] total articles from page 1: 10
> [scraping] total articles from page 2: 10
> [scraping] all articles saved at: out/articles-pemilu-2024-2-page.csv
> [scraping] done

```
